Detailed Description

template<class F>
class amrex::GShopLSFactory< F >

Template class that handles filling implicit functions from GeometryShop. The implicit function is filled into a MultiFab with the same CellSize, BoxArray, DistributionMapping, and number of ghost cells as the LSFactory.
