13 //
14 // A Collection of IArrayBoxes
15 //
16 // The iMultiFab class is publically derived from the
17 // FabArray<int,IArrayBox> class. It is a collection (stored as an array) of
18 // IArrayBoxes useful for storing integer data on a domain defined by
19 // a union of rectangular regions embedded in a uniform index space. The
20 // iMultiFab class extends the function of the underlying FabArray class just
21 // as the IArrayBox class extends the funtion of BaseFab<int>. Additional
22 // member functions are defined for I/O and simple arithmetic operations on
23 // these aggregate objects.
24 //
25 // This class does NOT provide a copy constructor or assignment operator.
26 //
