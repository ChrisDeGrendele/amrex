10 /*
11  A Mask is a derived Fab class whose elements maintain a logical
12  "TRUE" or "FALSE" state. A Mask can perform several elementary
13  logical operations on its data members.
14 
15  Implement a logical class Mask to maintain the TRUE or FALSE state
16  of elements in a box. A Mask can perform element-wise logical
17  operations, but is otherwise very similar to an FArrayBox object.
18  A Mask object can set its components based on comparing (a subregion
19  of a) real-valued FArrayBox element magnitudes to a scalar value, or
20  by "and", "or" operations with (the a subregion of) other Masks.
21 
22  This class does NOT provide a copy constructor or assignment operator.
